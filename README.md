# Hotel MBalu - Sistema de Reservas

## Descrição
Este projeto é um **sistema de agendamento de quartos de hotel**, desenvolvido com **HTML, CSS e JavaScript puro**. Permite ao usuário reservar quartos selecionando tipo, datas de check-in e check-out.  
Funcionalidades adicionais:
- Cálculo automático do total de acordo com o número de noites e tipo de quarto  
- Armazenamento de reservas no navegador (localStorage)  
- Remoção de reservas  
- Botão de contato via WhatsApp  

O objetivo é demonstrar habilidades em **lógica, interatividade e experiência de usuário**.

---

## Funcionalidades
- Seleção de quarto com preço por noite  
- Escolha de check-in e check-out  
- Cálculo automático do valor total  
- Visualização da lista de reservas  
- Remoção de reservas existentes  
- Botão WhatsApp para contato rápido

---

## Tecnologias
- HTML5  
- CSS3  
- JavaScript (ES6)  
- LocalStorage (persistência de dados no navegador)
